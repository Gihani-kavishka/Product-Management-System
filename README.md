# Product Management System

A simple **Spring Boot CRUD web application** developed for managing product details.  
This system allows users to add, view, update, and delete product records through a user-friendly web interface.

## Features
- Add new products
- View all products
- Update existing product details
- Delete products
- Simple and clean UI
- Database connectivity with MySQL

## Technologies Used
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- HTML
- CSS
- Bootstrap

## Project Structure
- `Controller` - Handles web requests
- `Entity` - Contains product model class
- `Repository` - Database access layer
- `Service` - Business logic layer
- `templates` - Thymeleaf frontend pages
- `application.properties` - Configuration file

## Screens Included
- Home Page
- Product List Page
- Add Product Page
- Edit Product Page
- Delete Product Page

## How to Run the Project

### Prerequisites
Make sure you have installed:
- Java JDK
- Maven
- MySQL
- Spring Tool Suite / Eclipse / IntelliJ
